# Chevrolet / GM OBD-II Diagnostic Trouble Codes (DTC)

## Code Prefix Categories
* Pxxxx – Powertrain (engine, transmission)
* Bxxxx – Body systems
* Cxxxx – Chassis systems
* Uxxxx – Network / communication systems
