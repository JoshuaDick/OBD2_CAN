# Chevrolet / GM CAN Bus Signal Reference

## Powertrain

- ID 0x0C9: Engine RPM (ECM)
- ID 0x0AA: Throttle Position (ECM)
- ID 0x0B4: Accelerator Pedal Position (ECM)
- ID 0x0C5: Engine Load (ECM)
- ID 0x0F1: Intake Air Temperature (ECM)
- ID 0x0F2: Coolant Temperature (ECM)

---

## Vehicle Motion

- ID 0x1F5: Vehicle Speed (ABS)
- ID 0x1A1: Steering Angle (Steering Control Module)
- ID 0x1A5: Yaw Rate (Stability Control)
- ID 0x1A6: Lateral Acceleration (Stability Control)

---

## Brake System

- ID 0x224: Brake Pedal Position (BCM)
- ID 0x225: Brake Pressure (ABS)

---

## Transmission

- ID 0x2A1: Current Gear (TCM)
- ID 0x2A2: Transmission Oil Temperature (TCM)

---

## Body Signals

- ID 0x3D1: Door Status (BCM)
- ID 0x3D2: Headlight Status (BCM)
- ID 0x3D3: Turn Signal Status (BCM)
- ID 0x3D4: Wiper Status (BCM)

---

## Driver Controls

- ID 0x4A1: Steering Wheel Buttons
- ID 0x4A2: Cruise Control Status

---

## Fuel System

- ID 0x5B1: Fuel Level (BCM)
- ID 0x5B2: Fuel Pump Status (ECM)

---

## HVAC

- ID 0x6A1: Cabin Temperature
- ID 0x6A2: Fan Speed

---

## Security

- ID 0x7E1: Immobilizer Status (BCM)
- ID 0x7E8: Diagnostic Response (ECM)
